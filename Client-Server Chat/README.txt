To compile the client, type "make" or "make all".


Run the server by typing: python chatserve.py [PORT NUMBER]


Note, you may need to give chatserve.py executable permission with: chmod +x chatserve.py


To get the server IP, in terminal running server, use: hostname -I



Run the client by typing: ./client [SERVER IP] [SERVER PORT]

First message must be sent by the client with alternating responses afterward.




Example run:

make

chatserve.py 35353

./client 128.193.36.41 35353
